A mootools based image carousel. This is very lightweight and extremely configurable pluggin. Supports both horizontal and vertical carousels with multiple options. You can change the background color of the carousel, the animation style, animation duration, arrow image themes etc. (Look for options in the end)

Tested with FF 3.6.11, IE 9.0.7 Beta, Chrome 7.0.517.41, Opera 10.60

Horizontal carousel

new mooCarousel({elementClass: '.imagesP',
		 slideStep: 2, 
		 elementsPerScreen: 3,
		 direction: 'horizontal',
		 bgColor: '#0D2247',
		 arrowColorScheme: 'darkblue'
		});

Vertical carousel


new mooCarousel({elementClass: '.imagesV',
		 elementsPerScreen: 3,
		 direction: 'vertical',
		 bgColor: '#104033',
		 arrowColorScheme: 'darkgreen'
		});

Option

Type

Default

Description
elementClass String None Name of the class which is applied on all images
elementsPerScreen Number 4 Number of images you want to display per carousel slide
slideStep Number 1 Number of images you want to move when carousel slide is moved using arrow buttons
durationTime String/Number 'long' Duration of the effect. Possible values are 'short', 'normal', 'long' or time in milliseconds
transitionEffect String 'Fx.Transitions.Bounce.easeOut' Mootools Fx.Transition effect. Effect using which the carousel should slide. Detail of Mootools Fx.Transition can be found here
bgColor String '#000000' (black) Background color of the carousel
direction String 'horizontal' Direction of the carousel. Possible values are "horizontal" and "vertical"
arrowColorScheme String 'blue' The color scheme of the arrow images. Possible values are "blue", "darkblue", "black", "lightgreen", "darkgreen", "orange", "red"
opacityLevel Number 0.4 When the carousel is moved, how much opacity of the images should be decreased. Possible values are 0 to 1
imagesPath String 'img/' The path where the arrow images are placed
  • Since, mooCarousel is a carousel script based on mootools, therefore, first you need to include mootools. You can download latest version from here.
  • Download the mooCarousel script and images from here and include it in the script
  • Since, the library requires all images to load, therefore, try a small trick of putting visibility:hidden in the style of all images. The plugin will itself change the visibility of images accordingly.
  • Assign all the elements same class name on which you want to create carousel
  • Instantiate the carousel class onload using;
    new mooCarousel({elementClass: '.imagesP',
		 slideStep: 2, 
		 elementsPerScreen: 3,
		 direction: 'horizontal',
		 bgColor: '#0D2247',
		 arrowColorScheme: 'darkblue'
		});
