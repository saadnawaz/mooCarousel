mooCarousel

A mootools based image carousel. This is very lightweight and extremely configurable pluggin. Supports both horizontal and vertical carousels with multiple options. You can change the background color of the carousel, the animation style, animation duration, arrow image themes etc. (Look for options in the end) Tested with FF 3.6.11, IE 7.0, Chrome 7.0.517.41, Opera 10.60

How to use

  • Since, mooCarousel is a carousel script based on mootools, therefore, first you need to include mootools. You can download latest version from here.
  • Download the mooCarousel script and images from here and include it in the script
  • Since, the library requires all images to load, therefore, try a small trick of putting visibility:hidden in the style of all images. The plugin will itself change the visibility of images accordingly.
  • Assign all the elements same class name on which you want to create carousel
  • Instantiate the carousel class onload using;
    new mooCarousel({elementClass: '.imagesP', elementsPerScreen: 3, direction: 'horizontal', bgColor: 'black', arrowColorScheme: 'blue'});
    • elementClass: null, slideStep: 1, opacityLevel: 0.5, elementsPerScreen: 4, bgColor: 'gray', direction: 'horizontal', transitionEffect: Fx.Transitions.Bounce.easeOut, durationTime: 'normal', arrowColorScheme: 'blue'
  • Options are:
    • elementsPerScreen: Number of images you want to display per carousel slide. Default to 4
    • elementClass: Name of the class which is applied on all images
    • slideStep: Number of images you want to move when carousel slide is moved using arrow buttons. By default 1 image is moved out of the carousel window
    • durationTime: Duration of the effect. Possible values are 'short', 'normal', 'long' or time in milliseconds. Default to 'long'
    • transitionEffect: Mootools Fx.Transition effect. Defaults to Fx.Transitions.Bounce.easeOut
    • bgColor: Background color of the carousel. Default is black (#000000)
    • direction: Direction of the carousel. Possible values are "horizontal" and "vertical". Default is "horizontal"
    • direction: Direction of the carousel. Possible values are "horizontal" and "vertical". Default is "horizontal"
    • arrowColorScheme: The color scheme of the arrow images. Possible values are "blue", "darkblue", "black", "lightgreen", "darkgreen", "orange", "red". Default is "blue"
    • opacityLevel: When the carousel is moved, how much opacity of the images should be decreased. Possible values are 0 to 1. Default is 0.4
